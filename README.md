# Linux Filesystem Automation Script

A Bash script to automate filesystem monitoring and cleanup workflow.

## Features

- Threshold-based trigger
- Old file detection
- Safe movement to archive directory
- Target filesystem safety check
- Logging support

## Workflow

Check filesystem usage  
↓  
Threshold validation  
↓  
Find old files  
↓  
Move files safely  
↓  
Generate logs

## Tech Used

- Linux
- Bash Scripting
- Filesystem Monitoring
- Automation Logic
